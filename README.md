# Parking Backend 🅿️

## 📌 Description
Backend for a **Parking Management System** built with **Node.js and TypeScript**.  
Manages parking slots, bookings, and feedback efficiently.  

## 🚀 Features
- RESTful API endpoints ⚡  
- TypeScript for type safety ✅  
- Database integration for parking data 💾  

## 🛠 Tech Stack
- Node.js, TypeScript, Express.js 💻  

## ⚙️ Installation
1. Clone repo: `git clone https://github.com/MuznaAli30/ParkingBackend.git`  
2. Install dependencies: `npm install`  
3. Start server: `ts-node server.ts`  
Server runs at http://localhost:5000 🌐
